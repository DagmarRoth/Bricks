# Running Into Brick

A data journalism investigation into brick-related injuries affecting American children using hospital emergency room data.

## Project Overview

This project analyzes National Electronic Injury Surveillance System (NEISS) data to uncover patterns in brick-related injuries among children under 18. The analysis reveals when, where, and how children get hurt colliding with brick structures.

## Files

- **brick_injuries_pudding_style.html** - Professional data journalism article with embedded visualizations (Pudding-style design)
- **brick_injuries_story.html** - NYT-style narrative with data insights
- **kids_brick_fun.html** - Playful visualization with interactive charts
- **Untitled-1.ipynb** - Jupyter notebook with full analysis and visualization code
- **hospital_visits.csv** - Full dataset of hospital visits
- **hospital_visits_sample.csv** - Sample dataset for quick reference

## Data Source

National Electronic Injury Surveillance System (NEISS) - A nationally representative sample of hospital emergency departments across the United States operated by the U.S. Consumer Product Safety Commission.

## Key Findings

- **{brick_count:,}** children visited hospitals due to brick-related injuries in a single year
- Children ages **6-12** experience the highest rate of brick collisions
- **Head and face injuries** are the most common type
- **Homes** are the most common location for brick injuries
- Older children suffer **more severe injuries** despite lower incident rates
